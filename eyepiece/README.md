# Eyepiece subsystem

Eyepiece-side adapters and accessories for the 1.25" focuser.

## Parts

| Part | What it is | Status |
|------|-----------|--------|
| [`microscope-adapter/`](microscope-adapter/) | Adapts a 1" microscope eyepiece (WF 10x/20) to the 1.25" focuser, with a female **M28.5 × 0.6** filter thread at the bottom. | v1 |

*Future candidates: parfocal rings, a 2" adapter, other eyepiece adapters.*
